# ExpressBlog-Content-Management-System
Description
Built a comprehensive blogging platform with Role-Based Access Control (RBAC), differentiating between standard authors and administrative moderators.

Engineered an advanced Search & Filter engine enabling users to discover content by title, category, or author name.

Implemented secure content management workflows, including image upload support, content previews with "Read More" functionality, and auto-hide flash notifications for user feedback.

Integrated an Admin Dashboard allowing for global content moderation, including the ability to edit or delete posts from any user to maintain platform integrity.
